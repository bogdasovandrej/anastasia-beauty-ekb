# Beauty Website - Богдашова Анастасия Викторовна

Красивый одностраничный сайт для beauty-мастера с онлайн-записью клиентов.

## 🎯 Особенности

- ✨ Минималистичный дизайн в нежных тонах
- 📱 Адаптивный дизайн (mobile-first)
- 📅 Онлайн-запись с выбором времени
- 📧 Отправка заявок на email через Formspree
- 🗓️ Автоматическое добавление в Google Calendar
- 🎁 Анимации и поздравление для мамы

## 🚀 Быстрый старт

### 1. Откройте сайт локально

```powershell
cd beauty-website
python -m http.server 8000
```

Откройте в браузере: `http://localhost:8000`

### 2. Замените фото мамы

Скопируйте фото мамы в `images/hero-v99.jpg` и обновите ссылку в `index.html`.

Подробная инструкция: [`INSTRUCTION_IMAGES.md`](INSTRUCTION_IMAGES.md)

### 3. Настройте Formspree

Formspree — сервис для отправки форм на email.

**Ваш Formspree endpoint:** `https://formspree.io/f/maqarovr`

1. Зайдите на [formspree.io](https://formspree.io)
2. Зарегистрируйтесь (бесплатно)
3. Подтвердите email
4. Проверьте, что заявки приходят на ваш email

### 4. Задеплойте на Vercel

Подробная инструкция: [`VERCEL_DEPLOY_INSTRUCTION.md`](VERCEL_DEPLOY_INSTRUCTION.md)

**Быстрый способ:**
1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите "Add New" → "Project"
3. Перетащите папку `beauty-website` в интерфейс
4. Нажмите "Deploy"

### 5. Купите домен anastasia-beauty-ekb.ru

**Цена:** ~169₽ в год (со скидкой)

Подробная инструкция: [`STEP_BY_STEP_EKB.md`](STEP_BY_STEP_EKB.md)

## 🐛 Возможные проблемы

### Картинки не загружаются

1. Проверьте, что файлы изображений находятся в папке `images/`
2. Убедитесь, что имена файлов совпадают (регистр важен!)
3. Проверьте пути в `index.html`
4. Если картинки закешированы, добавьте `?v=99` к URL

### Форма не отправляется

1. Проверьте, что Formspree endpoint правильный
2. Убедитесь, что email подтверждён в Formspree
3. Проверьте консоль браузера на ошибки

### Сайт медленно грузится

1. Оптимизируйте изображения (сжатие JPG/WebP)
2. Проверьте размер файлов (желательно < 200KB)
3. Используйте кэширование в `vercel.json`
