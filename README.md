### 2. Настройте Formspree

Formspree — сервис для отправки форм на email без backend-кода.

**Шаги:**

1. Зайдите на [formspree.io](https://formspree.io)
2. Зарегистрируйтесь (бесплатно)
3. Создайте новую форму (New Form)
4. Укажите email для получения заявок
5. Скопируйте URL формы (выглядит как `https://formspree.io/f/xv34abcd`)
6. Откройте `js/main.js` и замените:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/YOUR_FORMSPREE_ID';
   ```
   на ваш URL, например:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/xv34abcd';
   ```

### 3. Замените изображения

В `index.html` найдите ссылки на изображения и замените их на свои:

```html
<!-- Hero-фото -->

<!-- Фото услуг -->
<img src="images/manicure.jpg" alt="Маникюр" class="service-image">
```

**Где взять фото:**

  - [Pexels](https://pexels.com)

## ⚙️ Настройка услуг

В файле `js/main.js` найдите объект `serviceDurations`:

```javascript
const serviceDurations = {
    'manicure': 120,    // Маникюр - 2 часа
    'pedicure': 150,    // Педикюр - 2.5 часа
    'brows': 60,        // Брови - 1 час
    'lashes': 90,       // Ресницы - 1.5 часа
    'makeup': 120       // Визаж - 2 часа
};
```

Измените длительность услуг по необходимости.

## 🐛 Возможные проблемы

### Форма не отправляется
1. Проверьте, что настроен Formspree
2. Убедитесь, что URL формы правильный
3. Проверьте консоль браузера на ошибки

### Изображения не отображаются
1. Проверьте пути к файлам
2. Убедитесь, что файлы существуют
3. Проверьте имена файлов (регистр важен)

### Время не генерируется
1. Выберите услугу
2. Выберите дату
3. Проверьте консоль на ошибки
