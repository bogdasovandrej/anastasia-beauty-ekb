# Beauty Website - Богдашова Анастасия Викторовна

Красивый одностраничный сайт для beauty-мастера с онлайн-записью клиентов.

## 🎯 Особенности

- ✨ Минималистичный дизайн в нежных тонах
- 📱 Адаптивный дизайн (mobile-first)
- 📅 Онлайн-запись с выбором времени
- 📧 Отправка заявок на email через Formspree
- 🗓️ Автоматическое добавление в Google Calendar
- 🎁 Анимации и поздравление для мамы

## 🚀 Быстрый старт

### 1. Откройте сайт локально

Просто откройте файл `index.html` в браузере:

```bash
# Если у вас есть Python
cd beauty-website
python -m http.server 8000

# Затем откройте http://localhost:8000 в браузере
```

### 2. Настройте Formspree

Formspree — сервис для отправки форм на email без backend-кода.

**Шаги:**

1. Зайдите на [formspree.io](https://formspree.io)
2. Зарегистрируйтесь (бесплатно)
3. Создайте новую форму (New Form)
4. Укажите email для получения заявок
5. Скопируйте URL формы (выглядит как `https://formspree.io/f/xv34abcd`)
6. Откройте `js/main.js` и замените:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/YOUR_FORMSPREE_ID';
   ```
   на ваш URL, например:
   ```javascript
   const formspreeUrl = 'https://formspree.io/f/xv34abcd';
   ```

### 3. Замените изображения

В `index.html` найдите ссылки на изображения и замените их на свои:

```html
<!-- Hero-фото -->
<img src="images/hero.jpg" alt="Анастасия Викторовна" class="hero-photo">

<!-- Фото услуг -->
<img src="images/manicure.jpg" alt="Маникюр" class="service-image">
```

**Где взять фото:**
- Используйте предоставленные фото мамы
- Для услуг можно использовать бесплатные стоки:
  - [Unsplash](https://unsplash.com)
  - [Pexels](https://pexels.com)

### 4. Задеплойте на Vercel (бесплатно)

**С GitHub:**

1. Создайте аккаунт на [GitHub](https://github.com)
2. Создайте новый репозиторий
3. Загрузите файлы сайта в репозиторий
4. Зайдите на [vercel.com](https://vercel.com)
5. Войдите через GitHub
6. Нажмите "New Project"
7. Выберите ваш репозиторий
8. Нажмите "Deploy"

**Без GitHub (просто перетаскивание):**

1. Зайдите на [vercel.com](https://vercel.com)
2. Создайте аккаунт
3. Нажмите "Add New" → "Project"
4. Перетащите папку `beauty-website` в интерфейс
5. Нажмите "Deploy"

**Результат:** Ваш сайт будет доступен по адресу:
```
your-site.vercel.app
```

## ⚙️ Настройка услуг

В файле `js/main.js` найдите объект `serviceDurations`:

```javascript
const serviceDurations = {
    'manicure': 120,    // Маникюр - 2 часа
    'pedicure': 150,    // Педикюр - 2.5 часа
    'brows': 60,        // Брови - 1 час
    'lashes': 90,       // Ресницы - 1.5 часа
    'makeup': 120       // Визаж - 2 часа
};
```

Измените длительность услуг по необходимости.

## 🎨 Настройка дизайна

В файле `css/styles.css` найдите переменные CSS:

```css
:root {
    --color-bg: #fdfdfd;           /* Фон */
    --color-accent: #f5d0d0;       /* Акцент */
    --color-accent-hover: #f0b8b8; /* Акцент при наведении */
    --color-text-primary: #333333; /* Основной текст */
    --color-text-secondary: #666666; /* Вторичный текст */
}
```

Измените цвета по вашему вкусу.

## 🔧 Технические детали

### Формы и валидация
- Все поля обязательны для заполнения
- Автоматическое форматирование телефона
- Валидация даты (нельзя выбрать прошлые даты)

### Генерация слотов времени
- Маникюр/Педикюр/Визаж: слоты каждые 2 часа
- Брови: слоты каждый час (XX:00 и XX:30)
- Ресницы: слоты каждые 1.5 часа

После отправки формы клиент может добавить запись в свой календарь:
- Название: "Запись на [услуга]"
- Дата и время: выбранное
- Детали: Имя и телефон клиента
- Локация: адрес мастера

## 🐛 Возможные проблемы

### Форма не отправляется
1. Проверьте, что настроен Formspree
2. Убедитесь, что URL формы правильный
3. Проверьте консоль браузера на ошибки

### Изображения не отображаются
1. Проверьте пути к файлам
2. Убедитесь, что файлы существуют
3. Проверьте имена файлов (регистр важен)

### Время не генерируется
1. Выберите услугу
2. Выберите дату
3. Проверьте консоль на ошибки
