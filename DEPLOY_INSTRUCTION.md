# 🚀 Инструкция по деплою на Vercel и покупке домена

## Часть 1: Деплой на Vercel

### Вариант А: Через GitHub (рекомендуется)

#### Шаг 1: Создайте репозиторий на GitHub

1. Зайдите на [github.com](https://github.com)
2. Нажмите **"+"** → **"New repository"**
3. Назовите репозиторий, например: `beauty-website`
4. Сделайте его **Public**
5. Нажмите **"Create repository"**

#### Шаг 2: Загрузите файлы сайта

Откройте терминал в папке `beauty-website` и выполните:

```bash
# Инициализация git
git init

# Добавление всех файлов
git add .

# Первая коммит
git commit -m "Initial commit: beauty website"

# Связь с удаленным репозиторием
git remote add origin https://github.com/ВАШ_НИКNAME/beauty-website.git

# Отправка в GitHub
git push -u origin main
```

Если не знаете как создать SSH ключ или настроить GitHub, сделайте так:

1. Скачайте [GitHub Desktop](https://desktop.github.com)
2. Откройте папку `beauty-website`
3. Нажмите **"Create a repository"**
4. Назовите `beauty-website` и нажмите **"Create repository"**
5. Нажмите **"Publish repository"**
6. Выберите **"Public"** и нажмите **"Publish repository"**

#### Шаг 3: Подключите к Vercel

1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите **"Sign Up"** → выберите **"Continue with GitHub"**
3. После регистрации нажмите **"Add New..."** → **"Project"**
4. Выберите репозиторий `beauty-website`
5. Нажмите **"Import"**
6. Настройки можно оставить по умолчанию:
   - **Framework Preset:** Static Sites
   - **Build Command:** (пусто)
   - **Output Directory:** `beauty-website`
7. Нажмите **"Deploy"**

**Готово!** Ваш сайт будет доступен по адресу:
```
https://beauty-website.vercel.app
```

---

### Вариант Б: Без GitHub (просто перетаскивание)

1. Зайдите на [vercel.com](https://vercel.com)
2. Зарегистрируйтесь (через GitHub или email)
3. Нажмите **"Add New..."** → **"Project"**
4. Перетащите папку `beauty-website` в интерфейс
5. Нажмите **"Deploy"**

**Готово!** Сайт будет доступен по случайному адресу, например:
```
https://beauty-website-abc123.vercel.app
```

## Часть 2: Покупка домена

### Где купить домен

| Платформа | Цена (год) | Особенности |
|-----------|------------|-------------|
| **reg.ru** | ~200-500₽ | Российский регистратор, удобная панель |
| **nic.ru** | ~200-500₽ | Российский регистратор |
| **ru-center** | ~200-500₽ | Крупный регистратор |

### Рекомендуемые имена доменов

```
anastasia-beauty-ekb.ru
master-krasoty-ekb.ru
anastasia-viktorovna.ru
beauty-anastasia.ru
```

### Как купить домен на reg.ru

1. Зайдите на [reg.ru](https://reg.ru)
2. Введите желаемое имя домена в поиск
3. Если свободно — нажмите **"Зарегистрировать"**
4. Заполните данные (можно без ИП/ООО для .ru домена)
5. Оплатите (~200-500₽ в год)
6. Домен активируется автоматически

## Часть 3: Подключение домена к Vercel

### Шаг 1: Получите DNS-серверы Vercel

1. Зайдите в панель Vercel
2. Выберите ваш проект
3. Перейдите в **"Settings"** → **"Domains"**
4. Нажмите **"Add Domain"**
5. Введите ваш домен (например, `anastasia-beauty-ekb.ru`)
6. Нажмите **"Add"**

Vercel покажет DNS-серверы:
```
ns1.vercel-dns.com
ns2.vercel-dns.com
```

### Шаг 2: Настройте DNS на reg.ru

1. Зайдите в личный кабинет на [reg.ru](https://reg.ru)
2. Перейдите в **"Мои домены"**
3. Выберите ваш домен
4. Нажмите **"DNS-серверы"**
5. Выберите **"Указать свои NS-серверы"**
6. Введите:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ```
7. Нажмите **"Изменить"**

### Шаг 3: Добавьте поддомен (опционально)

Чтобы сайт был доступен как `www.anastasia-beauty-ekb.ru`:

1. В Vercel: **"Settings"** → **"Domains"**
2. Добавьте `www.anastasia-beauty-ekb.ru`
3. В reg.ru: **"DNS-записи"** → добавьте CNAME:
   - **Имя:** www
   - **Значение:** cname.vercel-dns.com

### Шаг 4: Дождитесь активации

DNS-изменения могут занять от 1 часа до 48 часов. Обычно работает через 1-2 часа.

После активации:
1. В Vercel: **"Settings"** → **"Domains"**
2. Нажмите на домен
3. Включите **"Force HTTPS"**

## Часто задаваемые вопросы

### Q: Сайт не открывается по домену
**A:** Подождите до 48 часов для активации DNS. Проверьте через [whatsmydns.net](https://whatsmydns.net).

### Q: Форма не отправляется
**A:** Проверьте, что в HTML указан правильный action:
```html
<form action="https://formspree.io/f/maqarovr" method="POST">
```

### Q: Как изменить имя домена?
**A:** Купите новый домен и подключите его к Vercel. Старый можно удалить.

### Q: Можно ли бесплатно?
**A:** Да! Vercel бесплатный, домен ~200-500₽ в год.
