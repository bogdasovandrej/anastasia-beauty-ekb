# Настройка Vercel - удаление старых версий и кастомный домен

## Удаление старых версий проектов на Vercel

### Способ 1: Через Dashboard

1. Откройте https://vercel.com/dashboard
2. Найдите проект `anastasia-beauty-ekb`
3. Нажмите на три точки (⋯) справа от проекта
4. Выберите **Delete**
5. Подтвердите удаление

**Важно:** Это удалит весь проект полностью!

### Способ 2: Переименовать проект

1. Откройте https://vercel.com/dashboard
2. Нажмите на проект `anastasia-beauty-ekb`
3. Перейдите в **Settings** → **General**
4. Измените **Project Name** на что-то другое
5. Создайте новый проект с именем `anastasia-beauty-ekb`

### Рекомендация

Не удаляйте старые проекты, если они работают. Лучше:
- Оставьте их как есть
- Или переименуйте в `anastasia-beauty-ekb-old`

## Добавление кастомного домена

### Шаг 1: Добавьте домен в Vercel

1. Откройте проект `anastasia-beauty-ekb`
2. Перейдите **Settings** → **Domains**
3. Нажмите **Add**
4. Введите: `anastasia-beauty-ekb.ru`
5. Нажмите **Add**

### Шаг 2: Настройка DNS на reg.ru

Vercel предложит два варианта:

#### Вариант А: Использовать NS-серверы Vercel (рекомендуется)

1. Скопируйте NS-серверы из Vercel:
   ```
   ns1.vercel-dns.com
   ns2.vercel-dns.com
   ```

2. На reg.ru:
   - Войдите в панель управления
   - Найдите домен `anastasia-beauty-ekb.ru`
   - Перейдите в **DNS-управление**
   - Найдите раздел **NS-серверы**
   - Измените на NS-серверы Vercel
   - Сохраните

#### Вариант Б: Использовать A-записи

1. На reg.ru добавьте записи:

   ```
   Тип: A
   Имя: @
   Значение: 76.76.21.21
   
   Тип: A
   Имя: @
   Значение: 76.76.21.1
   
   Тип: A
   Имя: @
   Значение: 76.76.21.10
   
   Тип: A
   Имя: @
   Значение: 76.76.21.15
   
   Тип: CNAME
   Имя: www
   Значение: cname.vercel-dns.com
   ```

### Шаг 3: Ожидайте распространения DNS

- **Время:** 1-48 часов (обычно 1-4 часа)
- Статус в Vercel изменится с "Pending" на "Active"

### Шаг 4: HTTPS включится автоматически

После активации DNS:
- HTTPS включится через Let's Encrypt
- Это занимает 10-30 минут

## Проверка что все работает

### В Vercel Dashboard

1. **Settings** → **Domains**
2. Статус должен быть зеленым "Active"

### DNS Checker

1. Откройте https://dnschecker.org/
2. Введите: `anastasia-beauty-ekb.ru`
3. Выберите тип: **A**
4. Нажмите "Check"

### Командная строка

```cmd
nslookup anastasia-beauty-ekb.ru
```

## FAQ

### Вопрос: Можно ли удалить старые проекты?
**Ответ:** Да, но не обязательно. Они не влияют на новый проект.

### Вопрос: Сколько времени занимает настройка домена?
**Ответ:** Обычно 1-4 часа, максимум 48 часов.

### Вопрос: HTTPS включится автоматически?
**Ответ:** Да, через Let's Encrypt после активации DNS.

### Вопрос: Что если домен не активируется?
**Ответ:** 
1. Проверьте NS-серверы на reg.ru
2. Подождите еще
3. Проверьте через DNS Checker
4. Напишите в поддержку Vercel
