# Проверка сайта на мобильных устройствах (Android/iOS)

## Как проверить сайт на Android

### Способ 1: Через браузер на телефоне

1. **Убедитесь что сайт загружен на Vercel**
   - Откройте браузер на Android
   - Перейдите по адресу: `https://anastasia-beauty-ekb.ru`
   - Если домен еще не активен, используйте временную ссылку Vercel

2. **Что проверить:**
   - ✅ Все секции отображаются корректно
   - ✅ Фотографии загружаются
   - ✅ Кнопки нажимаются (минимальный размер 44px)
   - ✅ Форма записи работает
   - ✅ Навигация по секциям
   - ✅ Текст читается без зума

### Способ 2: Эмуляция в Chrome DevTools (на компьютере)

1. Откройте сайт локально: `http://localhost:8000`
2. Нажмите `F12` для открытия DevTools
3. Нажмите `Ctrl+Shift+M` (Toggle device toolbar)
4. Выберите устройство:
   - **Samsung Galaxy S20** (Android)
   - **Pixel 5** (Android)
   - **iPhone 12** (iOS)
5. Проверьте отображение

### Способ 3: Chrome Remote Desktop

1. Установите Chrome Remote Desktop на Android
2. Подключитесь к компьютеру
3. Откройте сайт в браузере

## Что уже оптимизировано для мобильных

### CSS оптимизации:

```css
/* Адаптивность для экранов до 375px */
@media (max-width: 375px) {
    .hero-title { font-size: 28px; }
    .section-title { font-size: 24px; }
    .service-content { padding: 16px; }
}

/* Touch-устройства */
@media (hover: none) and (pointer: coarse) {
    .btn { min-height: 44px; } /* Минимальный размер для тапа */
    .form-group input { font-size: 16px; } /* Предотвращает зум на iOS/Android */
}
```

### HTML мета-теги:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0, user-scalable=yes">
<meta name="theme-color" content="#d4a5a5">
<meta name="apple-mobile-web-app-capable" content="yes">
<meta name="mobile-web-app-capable" content="yes">
<meta name="format-detection" content="telephone=yes">
```

## Распространенные проблемы и решения

### Проблема: Текст слишком мелкий
**Решение:** Увеличить `font-size` в медиа-запросах

### Проблема: Кнопки не нажимаются
**Решение:** Добавить `min-height: 44px` и `padding`

### Проблема: Горизонтальная прокрутка
**Решение:** Проверить `max-width: 100%` для изображений

### Проблема: Форма не отправляется
**Решение:** Проверить HTTPS и CORS настройки Formspree

## Как добавить сайт на главный экран Android

1. Откройте сайт в Chrome на Android
2. Нажмите меню (три точки) → "Добавить на главный экран"
3. Сайт появится как приложение с иконкой

## Как добавить сайт на главный экран iOS (iPhone)

1. Откройте сайт в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Выберите "На экран «Домой»"
4. Сайт появится как приложение

## Локальная проверка

Запустить локальный сервер:

```bash
cd beauty-website
python -m http.server 8000
```

Затем открыть: `http://localhost:8000`
